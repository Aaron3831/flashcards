JavaScript (JS)
The high-level programming language used by all modern Web browsers.
Operators
A symbol that is used to perform an operation.
Variables
Containers for storing data values.
Variable naming conventions
Makes code easier to read by capitilising the first letter in each new word following the first.
Functions
A function is a JavaScript procedure that performs a task or calcuates a value.
Methods
Methods are the actions that can be performed on objects.
Arguments
The values passed to the function that are used to give a new return value.
Parameters
Variables that can hold arguments.
Return
The value given by a function when it would be called.
Chaining Methods
Performing more than one method on the same object in succession.
Strings
A data object that stores a series of characters.
Booleans
Data object that can only be true or false.
Undefined
The undefined property indicates that a variable has not been assigned a value.
NaN
Indicates that a value is not a legal number.
Escape
The escape character turns the next character into a string character.
Alert
Displays an alert box with a specified message and an OK button.
Comments
A portion of code that will not be executed, typically used to describe how code works.
jQuery
A JavaScript library used mainly to add interactivity to the front-end.
